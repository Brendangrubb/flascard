A quick reference and study guide of javascript terms and concepts

JavaScript(JS)

Also known as ECMAScript, it is the only programming language supported by all existing web browsers. It is used to add interactivity and make web pages more dynamic.

Library

A collection of functions, methods, and objects used to perform related actions.

jQuery

A javascript library of functions used to easily make webpages interactive and dynamic.

Functions

Reusable blocks of code with specific names, which can be called on to perform specific tasks.

Methods

Reusable blocks of code with a specific name. Methods are associated with objects and acts on the objects which they are passed, optionally with additional parameters.

Chaining Methods

Passing returned values from functions or methods into another function or method as an arguement.

Arguements

Values passed into the parameters of a function or method.

Parameters

Optional values which a function or method can perform actions upon.

Return

A statement with functions and methods which serves to make data explicitly avaliable outside of the operation.

Variables

Various data types given descriptive names for use in programing and operations. E.g. String, Int, Float, Boolean, null.

Strings

A data type consisting of characters or symbols in sequence.

Numbers

A data type consisting of integer and floating point number values.

Booleans

A data type consisting of either True or False.

Undefined

A variable with no defined value.

Null

An intentional absence of any object value.

NaN

Any data in the number data type that does not conform to the conventions of that type, but is stored as a number value.

Infinity

The result of division by zero in javascript, stored as a number value.

Attributes

Values assigned to html elements to make them easily referenced and manipulated by javascript and css.

Operators

Symbols that indicate an action to be performed. E.g. +, -, *, /, %

Assignment Operators

Special symbols which denote an permenant action to be performed on a variable, changing its value.

Comparison Operators

Special symbols which denote comparisons to be performed between two or more values. Returns a boolean value.

Escape

Special characters used to denote special action to be taken.

Alert

A javascript function which creates a small window to alert the user with a specified message.

Comments

Specially escaped blocks of code with are not run normally in a program. They are used to inform readers of the code in question.
